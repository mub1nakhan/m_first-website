# Birinchi django app
## Qo'llanma

- birinchi o'zimizga kerakli papka yaratamiz
- virtual muhit yaratamiz
- virtual muhitni aktivlashtiramiz
- kerakli paketlarni o'rnatamiz

## Buyruqlar

Django project yaratish
```bash
django-admin startproject core .
```
Django app yaratish
```bash
python manage.py startapp main
```
Migratsiyalar yaratish 
```bash
python manage.py migrations
```
Migratsiyalarni amalga oshirish
```bash
python manage.py migrate
```
Serverni ishga tushurish
```bash
python manage.py runserver
```
